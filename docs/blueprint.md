# **App Name**: VideoCash

## Core Features:

- Video Ad Display: Display a list of available video ads with titles, rewards, and duration.
- Ad Watching and Reward: Allow users to watch ads for a specified duration, after which they receive a reward. After a video ad is complete a signal from the tool is generated so the funds are transferred.
- Balance Tracking: Display the user's current balance and track their progress towards the withdrawal goal.
- Referral System: Enable users to invite friends and earn a bonus for each referral.
- Withdrawal Functionality: Allow users to request payment once they reach the minimum withdrawal limit.
- Admin Panel: An admin panel that has sections for: analytics, manage ads, user management, and settings. It has an authentication screen and preconfigured credentials.
- Database Storage: Storing ads, balance, referrals to maintain information on videos watched per user.

## Style Guidelines:

- Primary color: Electric blue (#60a5fa) to represent digital currency and forward movement. Converted from HSL(213, 78%, 67%).
- Background color: Dark navy (#030712) to create a sense of depth and sophistication. Converted from HSL(220, 67%, 5%).
- Accent color: Violet (#c084fc) to create contrast and call attention to important elements, such as buttons or progress indicators. Converted from HSL(267, 67%, 76%).
- Body and headline font: 'Rajdhani' sans-serif for a modern, bold look.
- Use Font Awesome icons for a consistent and recognizable visual language.
- Use a glassmorphism effect with blurred backgrounds and subtle borders for a modern, immersive user experience.
- Implement subtle animations and transitions to enhance user engagement, such as progress bar updates or ad loading sequences.
